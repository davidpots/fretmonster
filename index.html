wisp

Lean & lightweight HTML/CSS/JS template.
Ideal for starting rapid prototypes.

Quick Start Guide

Here's how to quickly begin a new project using Wisp as a starting point:

  1. Download a copy of Wisp via GitHub or ZIP file
  2. In your local Wisp directory, open index.html and start coding.

That's it! It's that easy. Repeat every time you want to start fresh with a new idea.

Why Bother with Wisp?

Wisp is dead-simple and intentionally lightweight. It is intended to let you get an index.html file up and running as soon as possible, with pre-wired links to CSS and JS files. Benefits include:

  • Directories for images, CSS, and Javascript are already created
  • Your index.html is pre-wired to the usual CSS and jQuery resources, including essential meta tags and doctype info
  • Includes basic CSS for basic HTML typographic elements and scaffolding, including a CSS reset

If you're looking for something more robust and comprehensive, I recommend the HTML5 Boilerplate project.

Check Out the Sandbox

To get a glimpse of the HTML typographic elements that Wisp uses out of the gate, check out the Wisp Sandbox. This shows how basic HTML elements will appear on initial setup -- and then you can edit to your heart's content.
